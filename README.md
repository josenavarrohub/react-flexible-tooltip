## Prerequisites

You need to have [Node.js](https://nodejs.org/en/) 20.11.1 installed on your system.

## Installation

It is only necessary to clone the repository, install the dependencies and start the application with pnpm.

Dependencies:
```
pnpm install
```

Start application:
```
pnpm run dev
```
